# README.md

# SELLORY

Sellory adalah platform e-commerce modern berbasis SaaS untuk UMKM Indonesia.

Dibangun menggunakan:

* Frontend: HTML, CSS, JavaScript
* Backend: FastAPI (Python)
* Database: Supabase PostgreSQL
* Authentication: JWT
* Storage: Supabase Storage

---

# Vision

Membantu UMKM Indonesia memiliki toko online premium dengan dashboard modern, automation, dan AI tools.

---

# Main Features

* Multi-theme storefront
* Product management
* Order management
* Analytics dashboard
* AI-powered tools
* WhatsApp integration
* Payment integration
* Store customization
